# Vierkant Escape Room

A browser-based escape room game built with Phaser 3 and TypeScript. Players crash-land on a planet and must solve puzzles to escape.

**Target Audience:** Dutch elementary school children (grades 6-8)
**Tech Stack:** Phaser 3, TypeScript, Vite

---

## Installation

```bash
npm install
```

---

## Development

### Run locally (desktop only)
```bash
npm run dev
```
Opens at `http://localhost:5173`

### Run with mobile device testing
```bash
npm run dev:mobile
```
Opens dev server on your local network. The terminal will display IP addresses you can use to access the game from your phone.

**Note:** Anyone on your WiFi network can access the dev server. Only people outside your network cannot reach it.

### Build for production
```bash
npm run build
```
